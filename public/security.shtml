Security

Last Updated: January 19, 2025

At Yool.cl, we implement the following measures to ensure security:

    Use of HTTPS protocols to protect data transmission.
    Continuous monitoring to detect vulnerabilities.
    Restricted access to sensitive information.

If you detect any security issues, contact us immediately at seguridad@yool.cl.

Seguridad

Última Actualización: 19 de enero de 2025

En Yool.cl implementamos las siguientes medidas para garantizar la seguridad:

    Uso de protocolos HTTPS para proteger la transmisión de datos.
    Monitoreo constante para detectar vulnerabilidades.
    Restricción de acceso a información sensible.

En caso de detectar algún problema de seguridad, contáctanos de inmediato en seguridad@yool.cl.
